数据库

备份与恢复

备份

mongodump -h 数据库主机地址 -d 数据库名 -o 路径

-h：数据库主机地址 -d：需要备份的数据库名称 -o：备份的数据存放位置，此目录中存放着备份出来的数据

mongodb备份在sell终端中执行，创建一个文件夹保存备份文件

mkdir mongodb_backup

将pytest库备份导mongodb_backup文件夹下

mongodump -h localhost -d pytest -o ~/mongod_backup

恢复

mongorestore -h dbhost -d dbname --dir dbdirectory

-h：服务器地址，如果不是本地恢复，可以用ip:端口
-d：需要恢复的数据库
--dir：备份数据所在位置

恢复数据库：

mongorestore -h localhost -d backup --dir ~/mongod_backup/pytest

使用IP加端口的方式

备份

mongodump -h 127.0.0.1:27017 -d mytest -o ~/mongod_backup

恢复

mongorestore -h 127.0.0.1:27017 -d  mytest_back --dir ~/mongod_backup/mytest
